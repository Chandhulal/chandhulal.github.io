# Blue Ice - Professional Bartending Services

A single-page website for Blue Ice bartending services by Saj MG.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Features

- Full-screen hero section with dark bartender background
- Service specialization cards with hover effects
- Responsive mobile-first design
- Clean, professional UI with Tailwind CSS
- Minimal animations for smooth user experience
